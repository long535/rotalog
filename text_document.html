# How to Download and Use Your Rotalog Project You can access the full code (HTML and Tailwind CSS) for each screen in the Rotalog project directly from this workspace. ### Accessing the Code 1. **Select a Screen:** Click on any of the screens in the canvas that you'd like to work with. 2. **View Code:** Once a screen is selected, you'll see an option to view or copy the underlying HTML/Tailwind CSS code. 3. **Download:** You can copy this code and save it into your local development environment. ### Project Structure Recommendation Since this project is designed using **React + Tailwind CSS**, here’s how I recommend organizing your local project: 1. **Initialize your React app:** ```bash npx create-react-app rotalog cd rotalog ``` 2. **Install Tailwind CSS:** Follow the official [Tailwind CSS installation guide for Create React App](https://tailwindcss.com/docs/guides/create-react-app). 3. **Create Components:** For each screen I've designed, create a new React component (e.g., `Home.js`, `Calendar.js`, `Settings.js`) and paste the corresponding code. 4. **Fonts:** The design uses the **Lexend** font. Ensure you include it in your `index.html` or CSS: ```html ``` ### Screens Included in Your Project: - **Home (Shift List):** Main dashboard with shift cards. - **Calendar View:** Monthly overview of shifts. - **Add/Edit Shift:** Bottom sheet modal for entry. - **Statistics Dashboard:** Visual charts for hours and earnings. - **Earnings Breakdown:** Detailed financial view. - **Tax Settings:** Configuration for rates and deductions. - **Export Tax Report:** Document generation and scheduling. - **Settings:** App preferences and theme selection.
